# FBI's Most Wanted

Web client for the FBI's public REST API. Next.js project created with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

TailwindCSS + daisyUI used for UI.

## Getting Started (Development)

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view app.
